# Project Overview

This repository contains a simple training setup used to practice basic Git
operations, file structure management, and version control workflow.

## Files
- hello.txt — sample text file
- README.md — project documentation

## Covered Concepts
- Initializing a repository
- Staging and committing changes
- Managing files through Git
- Working with branches
- Pushing updates to a remote repository

## Notes
The repository serves as a clean and minimal example of how a basic Git workflow
is organized.
